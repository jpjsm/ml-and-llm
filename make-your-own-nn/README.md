## Setup

- create project folder
- setup and activate virtual environment, install:
  - python=3.12
  - ipython=8.24.0 
  - jupyterlab=4.2.0 
  - numpy=1.26.4 
  - matplotlib=3.9.0
  - scipy 1.13.0
